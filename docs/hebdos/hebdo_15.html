Filtrer le jeu de données

Le masque établi a gardé quelques pixels qui peuvent être interprété par de la mer par les données Sentinel. On va donc filtrer à une altitude dépassant les 50m et un indice de NDVI élevé afin de ne garder seulement les forêts denses pré-cyclones (en décembre pour Chido).

par(mfrow = c(1,2))
hist(dta$altitude, breaks = 50, col = "lightblue",
     xlab = "Altitude", main = NULL)
abline(v = 50, col = "red", lty = 2)
hist(dta$dec_ndvi, breaks = 50, col = "lightgreen",
     xlab = "NDVI pré-cyclonique", main = NULL)
abline(v = 0.7, col = "red", lty = 2)
plot(dta$msw_max, dta$delta_ndvi)

On peut voir les pixels parasites aux alentours de 10m d’altitude. De plus, le NDVI de décembre est caractérisé par des forêts denses en majorité, on établit donc un seuil de 0.7. Cela nous permettra d’éliminer l’effet que l’on peut voir dans le troisième graphique où, lorsque le vent est plus fort, on a passé un certain seuil, une remontée du delta_ndvi.

dtaFiltered <- dta[(dta["dec_ndvi"] > 0.7) & (dta["altitude"] >= 50),]

par(mfrow = c(1,2))
hist(dtaFiltered$altitude, breaks = 50, col = "lightblue",
     xlab = "Altitude", main = "après filtre")
hist(dtaFiltered$dec_ndvi, breaks = 20, col = "lightgreen",
     xlab = "NDVI pre-cyclonique", main = "après filtre")

Ce filtrage devrait considérablement améliorer les modèles. De plus, il faut standardisé les différentes variables afin qu’elles soient comparables entre elles.

scaledVar <- c("altitude","wem","pente","courbure","ang6","msw_max")

dtaScaled[scaledVar] <- lapply(dtaScaled[scaledVar], scale)

Predictions VS Observées

On peut regarder les observations VS les prédictions des différents modèles sur le jeu de test afin de voir si les modèles prédisent correctement les dommages de végétation.

models <- list(m1 = m1, m2 = m2, m3 = m3, m4 = m4, m5 = m5)


par(mfrow = c(2, 3))

for (nom in names(models)) {
  m <- models[[nom]]
  pred <- predict(m, newdata = dta_test)
  obs  <- dta_test$delta_ndvi
  
  plot(obs, pred,
       main = paste(nom, "- Obs vs Pred"),
       xlab = "Observé", ylab = "Prédit",
       pch = 16, col = "steelblue")
  abline(0, 1, col = "red", lwd = 2)  # droite parfaite y = x
}

Résidus : Valeur observée - valeur prédite Vérifier l’autocorrélation spatiale des résidus dans un contexte spatial grâce au test de Moran.

Zones d’intérêts

Afin de mieux visualiser à plus petite échelle, on peut établir plusieurs zones d’intérêts :

  • Nord : Au Nord de l’île, ou le vent a frappé plutôt fort (max : 55.5 m/s) avec une altitude allant de 294 à 519m.
  • Sud : Au Sud de l’île, où le vent a frappé moins fort (max : 50 m/s) avec une altitude allant de 54 à 309m.
  • Mbouzi : L’ilôt Chissouia Mbouzi où le vent a eu un pic à 61 m/s et son altitude va jusqu’à 149m.
